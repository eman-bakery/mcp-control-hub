# MCP Control Hub

MCP Control Hub is a mobile-first, production-ready MCP platform for internal business tools and AI agents.

It acts as a central control panel where connectors can be enabled and managed for platforms like Supabase, GitHub, Vercel, Gmail, Outlook, and OneDrive, with support for adding more connectors later.

## Goals

- Provide a beautiful mobile-first web UI.
- Expose internal and external tools through a remote MCP server.
- Let AI agents securely use connected tools through MCP.
- Support modular connectors instead of hardcoding one giant integration.
- Keep deployment simple and cloud-based.
